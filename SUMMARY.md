## 🚀 Быстрый старт

### Для нетерпеливых (3 команды)

```bash
cd callbot
chmod +x install.sh
./install.sh
```

### Что нужно подготовить

1. **Telegram Bot Token**
   - Откройте @BotFather в Telegram
   - Команда: `/newbot`
   - Скопируйте токен

2. **Gemini API Key**
   - Откройте: https://makersuite.google.com/app/apikey
   - Создайте API ключ
   - Скопируйте ключ

3. **Ubuntu 22.04 сервер** (или локальный ПК)
   - Минимум 512 MB RAM
   - Docker установится автоматически

## 🎉 Готово к использованию!

Проект полностью готов к деплою на Ubuntu 22.04.

### Следующие шаги:

1. **Скопируйте проект на сервер**
   ```bash
   scp -r callbot user@server:/home/user/
   ```

2. **Подключитесь к серверу**
   ```bash
   ssh user@server
   cd callbot
   ```

3. **Запустите установку**
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

4. **Наслаждайтесь!**
   - Откройте Telegram
   - Найдите вашего бота
   - Отправьте `/start`
