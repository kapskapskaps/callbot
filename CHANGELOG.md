# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

## [1.1.0] - 2026-05-05

### Добавлено
- Анализ фекалий (копрограмма) по фото
- Оценка по Бристольской шкале стула
- Рекомендации по питанию при проблемах с пищеварением
- Новая кнопка "💩 Анализ стула" в меню
- Метод `analyze_stool()` в классе NutritionistAnalyzer

### Изменено
- Обновлена документация с описанием новой функции
- Расширено приветственное сообщение бота
- Обновлена команда /help с информацией об анализе стула

## [1.0.0] - 2026-05-05

### Добавлено
- Базовая функциональность бота-нутрициолога
- Анализ калорийности блюд по фото
- Анализ состава продуктов по фото этикетки
- Общая оценка блюд с рекомендациями
- Анализ медицинских анализов крови
- Интеграция с Google Gemini API
- Docker контейнеризация
- Подробная документация
- Скрипт автоматической установки
- Логирование в файл и консоль
- Обработка ошибок
- Inline клавиатура для выбора типа анализа

### Технические детали
- Python 3.11
- python-telegram-bot 20.8
- google-generativeai 0.8.0
- Docker и Docker Compose поддержка
- Автоматический перезапуск контейнера
- Ротация логов
