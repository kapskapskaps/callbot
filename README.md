# 🤖 Бот-Нутрициолог

Telegram-бот для анализа питания с использованием AI (Google Gemini API). Бот умеет анализировать фотографии еды, этикеток продуктов, медицинских анализов и фекалий, предоставляя профессиональные рекомендации по питанию.

## 🌟 Возможности

- **🍽 Оценка калорийности** - анализ калорий, БЖУ и веса порции по фото блюда
- **📋 Анализ состава** - расшифровка состава продукта по фото этикетки, выявление вредных добавок
- **🥗 Оценка блюда** - комплексный анализ сбалансированности питания и рекомендации
- **🩺 Анализ медицинских данных** - интерпретация результатов анализов крови с рекомендациями по питанию
- **💩 Анализ фекалий** - оценка по Бристольской шкале, выявление проблем с пищеварением, рекомендации по питанию

## 📋 Требования

- Ubuntu 22.04 (или другой Linux дистрибутив)
- Docker 20.10+
- Docker Compose 2.0+
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
- Google Gemini API Key (получить на [Google AI Studio](https://makersuite.google.com/app/apikey))

## 🚀 Быстрый старт

### 1. Клонирование репозитория

```bash
git clone <your-repo-url>
cd callbot
```

Или создайте директорию и скопируйте файлы проекта:

```bash
mkdir callbot
cd callbot
# Скопируйте все файлы проекта в эту директорию
```

### 2. Получение API ключей

#### Telegram Bot Token

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям: введите имя бота и username
4. Скопируйте полученный токен (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

#### Google Gemini API Key

1. Перейдите на [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Войдите с Google аккаунтом
3. Нажмите "Get API Key" → "Create API key"
4. Скопируйте созданный ключ

### 3. Настройка переменных окружения

Создайте файл `.env` на основе примера:

```bash
cp .env.example .env
nano .env
```

Заполните файл `.env`:

```env
# Telegram Bot Token (получить у @BotFather)
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz

# Gemini API Key (получить на https://makersuite.google.com/app/apikey)
GEMINI_API_KEY=AIzaSyABC123def456GHI789jkl

# Опционально: ID администраторов бота (через запятую)
ADMIN_IDS=123456789,987654321
```

**Важно**: Замените значения на ваши реальные токены!

### 4. Установка Docker и Docker Compose (если не установлены)

```bash
# Обновление системы
sudo apt update && sudo apt upgrade -y

# Установка Docker
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# Добавление пользователя в группу docker
sudo usermod -aG docker $USER

# Установка Docker Compose
sudo apt install docker-compose-plugin -y

# Перезагрузка для применения изменений группы
newgrp docker
```

Проверка установки:

```bash
docker --version
docker compose version
```

### 5. Запуск бота

```bash
# Сборка и запуск контейнера
docker compose up -d

# Просмотр логов
docker compose logs -f

# Остановка бота
docker compose down
```

## 🎯 Использование бота

### Команды

- `/start` - Приветствие и краткая информация
- `/help` - Подробная справка по использованию

### Как пользоваться

1. Найдите вашего бота в Telegram по username
2. Отправьте команду `/start`
3. Отправьте фото:
   - Блюда для оценки калорий
   - Этикетки продукта для анализа состава
   - Еды для общей оценки
   - Медицинских анализов
4. Выберите тип анализа из предложенного меню
5. Получите детальный ответ от AI

### Примеры использования

**Оценка калорийности:**
- Сфотографируйте тарелку с едой
- Выберите "🍽 Калории"
- Получите: калории, БЖУ, вес порции, рекомендации

**Анализ состава:**
- Сфотографируйте этикетку с составом
- Выберите "📋 Состав"
- Получите: расшифровку ингредиентов, вредные добавки, оценку полезности

**Оценка блюда:**
- Сфотографируйте готовое блюдо
- Выберите "🥗 Оценить блюдо"
- Получите: оценку сбалансированности, плюсы/минусы, рекомендации

**Анализ медицинских данных:**
- Сфотографируйте результаты анализов
- Выберите "🩺 Анализы"
- Получите: интерпретацию показателей, рекомендации по питанию


```bash

```

## 🐛 Решение проблем

### Бот не запускается

1. Проверьте правильность токенов в `.env`:
```bash
cat .env
```

2. Проверьте логи:
```bash
docker compose logs
```

3. Убедитесь, что порты не заняты:
```bash
docker compose ps
```

### Ошибки API

**"Invalid token"** - неверный Telegram Bot Token
- Проверьте токен в `.env`
- Получите новый токен у @BotFather

**"API key not valid"** - неверный Gemini API Key
- Проверьте ключ в `.env`
- Создайте новый ключ на Google AI Studio
- Убедитесь, что API включен в вашем Google Cloud проекте

### Бот не отвечает

1. Проверьте, что контейнер запущен:
```bash
docker compose ps
```

2. Перезапустите бота:
```bash
docker compose restart
```

3. Проверьте интернет-соединение на сервере

### Ошибки анализа изображений

- Убедитесь, что фото четкое и хорошего качества
- Проверьте, что на фото действительно есть еда/этикетка/анализы
- Попробуйте другое фото

## 🔄 Автозапуск при перезагрузке сервера

Бот настроен на автоматический запуск (`restart: unless-stopped` в docker-compose.yml).

Для проверки:

```bash
# Перезагрузка сервера
sudo reboot

# После перезагрузки проверьте статус
docker compose ps
```
