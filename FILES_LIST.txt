КОМАНДЫ ДЛЯ РАБОТЫ
------------------
Установка:
  chmod +x install.sh && ./install.sh

Запуск:
  docker compose up -d

Логи:
  docker compose logs -f

Остановка:
  docker compose down

Перезапуск:
  docker compose restart

Статус:
  docker compose ps
