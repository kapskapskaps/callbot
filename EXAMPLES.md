# Примеры использования бота

## 📸 Примеры запросов

### 1. Оценка калорийности

**Что отправить:**
- Фото готового блюда на тарелке
- Фото порции еды
- Фото перекуса

**Что получите:**
```
🍽 Анализ калорийности

Блюдо: Куриная грудка с гречкой и овощами

Примерный вес порции: 350 г

Пищевая ценность:
• Калории: 420 ккал
• Белки: 45 г
• Жиры: 8 г
• Углеводы: 42 г

Рекомендации:
✅ Отличный сбалансированный обед
✅ Высокое содержание белка
✅ Подходит для похудения и набора мышечной массы
⚠️ Можно добавить больше овощей для клетчатки
```

### 2. Анализ состава продукта

**Что отправить:**
- Фото этикетки с составом
- Фото обратной стороны упаковки
- Четкое фото списка ингредиентов

**Что получите:**
```
📋 Анализ состава продукта

Продукт: Йогурт фруктовый

Основные ингредиенты:
1. Молоко нормализованное
2. Сахар
3. Фруктовый наполнитель (клубника 8%)
4. Загуститель (E1442)
5. Консервант (E202)
6. Краситель (E120)

Пищевая ценность на 100г:
• Калории: 95 ккал
• Белки: 3.2 г
• Жиры: 2.5 г
• Углеводы: 15.4 г (из них сахара: 14 г)

⚠️ Вредные добавки:
• E1442 - модифицированный крахмал
• E202 - сорбат калия (консервант)
• E120 - кармин (краситель из насекомых)

Оценка полезности: 4/10

Рекомендации:
❌ Высокое содержание сахара
❌ Наличие искусственных добавок
✅ Содержит белок
⚠️ Не рекомендуется детям и людям с аллергией
💡 Лучше выбрать натуральный йогурт без добавок
```

### 3. Оценка блюда

**Что отправить:**
- Фото завтрака/обеда/ужина
- Фото домашней еды
- Фото ресторанного блюда

**Что получите:**
```
🥗 Оценка блюда

На фото: Овсяная каша с бананом, орехами и медом

Оценка сбалансированности: 8/10

Плюсы:
✅ Сложные углеводы для энергии
✅ Клетчатка для пищеварения
✅ Полезные жиры из орехов
✅ Калий и витамины из банана

Минусы:
⚠️ Много быстрых углеводов (банан + мед)
⚠️ Недостаточно белка

Рекомендации по улучшению:
• Добавить греческий йогурт или творог (белок)
• Уменьшить количество меда
• Добавить ягоды вместо части банана
• Посыпать семенами чиа или льна

Подходит для: Завтрака
Рекомендуется: Спортсменам, активным людям
Стоит избегать: Людям с диабетом (много сахара)
```

### 4. Анализ медицинских данных

**Что отправить:**
- Фото результатов общего анализа крови
- Фото биохимического анализа
- Фото анализа на гормоны

**Что получите:**
```
🩺 Анализ медицинских данных

Тип анализа: Общий анализ крови

Показатели:

✅ В норме:
• Эритроциты: 4.5 × 10¹²/л (норма: 4.0-5.5)
• Лейкоциты: 6.2 × 10⁹/л (норма: 4.0-9.0)
• Тромбоциты: 250 × 10⁹/л (норма: 180-320)

⚠️ Ниже нормы:
• Гемоглобин: 115 г/л (норма: 120-160)
• Железо: 8 мкмоль/л (норма: 10-30)

Что это означает:
Пониженный гемоглобин и железо указывают на железодефицитную анемию.

Рекомендации по питанию:

Включить в рацион:
✅ Красное мясо (говядина, телятина) - 3-4 раза в неделю
✅ Печень (говяжья, куриная) - 1-2 раза в неделю
✅ Гречка, чечевица, фасоль
✅ Яйца (желток)
✅ Гранат, яблоки, курага
✅ Зелень (шпинат, петрушка)
✅ Витамин C для лучшего усвоения железа (цитрусовые)

Ограничить:
❌ Чай и кофе во время еды (мешают усвоению железа)
❌ Молочные продукты вместе с железосодержащими

Режим питания:
• 4-5 приемов пищи в день
• Обязательный завтрак с белком
• Перекусы с орехами и сухофруктами

⚠️ ВАЖНО: Обратитесь к врачу-гематологу для назначения препаратов железа!
Эти рекомендации дополняют, но не заменяют медикаментозное лечение.
```

## 💡 Советы для лучших результатов

### Качество фото

✅ **Хорошо:**
- Четкое изображение
- Хорошее освещение
- Фото сверху для еды
- Вся этикетка в кадре
- Текст анализов читаемый

❌ **Плохо:**
- Размытое фото
- Темное освещение
- Часть информации обрезана
- Текст не читается

### Типичные ошибки

1. **Слишком общее фото** - старайтесь фотографировать конкретное блюдо, а не весь стол
2. **Плохой ракурс** - для еды лучше фото сверху
3. **Неполная этикетка** - убедитесь, что весь состав в кадре
4. **Отражения на этикетке** - избегайте бликов от вспышки

## 🔄 Типичный сценарий использования

### Сценарий 1: Контроль калорий

```
Пользователь → Фото завтрака
Бот → Меню выбора
Пользователь → "🍽 Калории"
Бот → Анализ: 450 ккал, БЖУ, рекомендации

Пользователь → Фото обеда
Бот → Меню выбора
Пользователь → "🍽 Калории"
Бот → Анализ: 650 ккал, БЖУ, рекомендации

Итого за день: ~1100 ккал (+ ужин)
```

### Сценарий 2: Выбор продуктов в магазине

```
Пользователь → Фото этикетки йогурта #1
Бот → Меню выбора
Пользователь → "📋 Состав"
Бот → Оценка 4/10, много сахара и добавок

Пользователь → Фото этикетки йогурта #2
Бот → Меню выбора
Пользователь → "📋 Состав"
Бот → Оценка 8/10, натуральный состав

Решение: Покупаем йогурт #2
```

### Сценарий 3: Планирование питания по анализам

```
Пользователь → Фото анализов крови
Бот → Меню выбора
Пользователь → "🩺 Анализы"
Бот → Низкий гемоглобин, рекомендации по питанию

Пользователь → Фото блюда с печенью
Бот → Меню выбора
Пользователь → "🥗 Оценить блюдо"
Бот → Отлично для повышения железа!
```

## 📱 Интеграция в повседневную жизнь

### Утро
- Фото завтрака → оценка калорий
- Планирование дневного рациона

### День
- Фото обеда → проверка сбалансированности
- Выбор продуктов в магазине → анализ составов

### Вечер
- Фото ужина → контроль калорий
- Анализ дневного рациона

### Периодически
- Загрузка новых анализов крови
- Корректировка питания по результатам

## ⚠️ Ограничения

Бот НЕ может:
- Поставить медицинский диагноз
- Заменить консультацию врача
- Гарантировать 100% точность калорий (оценка приблизительная)
- Учесть индивидуальные особенности организма
- Назначить лечение

Бот МОЖЕТ:
- Дать общие рекомендации по питанию
- Помочь с подсчетом калорий
- Расшифровать состав продуктов
- Указать на отклонения в анализах
- Предложить корректировку рациона
