## 🚀 Как начать использовать

### Шаг 1: Получите API ключи

**Telegram Bot Token:**
1. Откройте @BotFather в Telegram
2. Отправьте `/newbot`
3. Скопируйте токен

**Gemini API Key:**
1. Откройте https://makersuite.google.com/app/apikey
2. Создайте API ключ
3. Скопируйте ключ

### Шаг 2: Деплой на Ubuntu 22.04

```bash
# Перейдите в директорию проекта
cd callbot

# Запустите автоматическую установку
chmod +x install.sh
./install.sh

# Введите ваши API ключи когда скрипт попросит
```

### Шаг 3: Проверьте работу

```bash
# Проверьте статус
docker compose ps

# Посмотрите логи
docker compose logs -f
```

### Шаг 4: Используйте бота

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Отправьте фото
5. Выберите тип анализа
6. Получите результат!
