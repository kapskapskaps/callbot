## Возможности бота

### 1. Анализ калорийности 🍽
- Определение калорий по фото
- Расчет БЖУ (белки, жиры, углеводы)
- Оценка веса порции
- Рекомендации по питанию

### 2. Анализ состава 📋
- Расшифровка ингредиентов
- Выявление вредных добавок (E-шки)
- Оценка полезности (1-10)
- Рекомендации: кому подходит/избегать

### 3. Оценка блюда 🥗
- Анализ сбалансированности
- Плюсы и минусы блюда
- Рекомендации по улучшению
- Подходящий прием пищи

### 4. Анализ медицинских данных 🩺
- Интерпретация анализов крови
- Выявление отклонений от нормы
- Рекомендации по питанию
- Какие продукты включить/исключить

## Технологический стек

### Backend
- **Python 3.11** - основной язык
- **python-telegram-bot 20.8** - Telegram Bot API
- **google-generativeai 0.8.0** - Google Gemini API
- **aiohttp 3.9.5** - асинхронные HTTP запросы
- **pillow 10.3.0** - обработка изображений
- **python-dotenv 1.0.1** - переменные окружения

### Infrastructure
- **Docker** - контейнеризация
- **Docker Compose** - оркестрация
- **Ubuntu 22.04** - целевая ОС

### AI
- **Google Gemini 1.5 Flash** - мультимодальная AI модель
- **Vision API** - анализ изображений
- **Natural Language Processing** - генерация текста

## Производительность

### Время отклика
- Получение фото: < 1 сек

- Отправка результата: < 1 сек
- **Общее время:** 6-17 секунд

### Ограничения API

  - ~45,000 запросов в месяц

### Потребление ресурсов
