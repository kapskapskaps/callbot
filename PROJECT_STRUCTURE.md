# Структура проекта

```
callbot/
├── bot.py                  # Основной файл бота с обработчиками
├── analyzers.py            # Модули анализа с Gemini API
├── config.py               # Конфигурация и переменные окружения
├── requirements.txt        # Python зависимости
├── Dockerfile              # Docker образ для контейнеризации
├── docker-compose.yml      # Docker Compose конфигурация
├── install.sh              # Скрипт автоматической установки
├── .env                    # Переменные окружения (создать вручную)
├── .env.example            # Пример файла .env
├── .gitignore              # Игнорируемые файлы для Git
├── LICENSE                 # Лицензия MIT
├── README.md               # Основная документация
├── QUICKSTART.md           # Быстрый старт
├── EXAMPLES.md             # Примеры использования
├── FAQ.md                  # Часто задаваемые вопросы
├── CHANGELOG.md            # История изменений
└── logs/                   # Директория для логов (создается автоматически)
    └── bot.log
```

## Описание файлов

### Основные файлы

**bot.py** (7.2 KB)
- Главный файл приложения
- Обработчики команд `/start`, `/help`
- Обработчик фотографий
- Inline клавиатура для выбора типа анализа
- Обработка callback'ов от кнопок
- Логирование и обработка ошибок

**analyzers.py** (5.5 KB)
- Класс `NutritionistAnalyzer`
- 4 метода анализа:
  - `analyze_calories()` - оценка калорийности
  - `analyze_composition()` - анализ состава
  - `analyze_food()` - общая оценка блюда
  - `analyze_medical_tests()` - анализ медицинских данных
- Интеграция с Google Gemini API
- Промпты для AI на русском языке

**config.py** (470 B)
- Загрузка переменных окружения из `.env`
- Валидация наличия обязательных токенов
- Экспорт конфигурации для других модулей

### Docker файлы

**Dockerfile** (548 B)
- Базовый образ: Python 3.11-slim
- Установка системных зависимостей
- Установка Python пакетов
- Копирование кода приложения
- Команда запуска бота

**docker-compose.yml** (331 B)
- Сервис `nutritionist-bot`
- Автоматический перезапуск
- Монтирование логов
- Настройка часового пояса
- Ротация логов (max 10MB, 3 файла)
