# 🚀 НАЧНИТЕ ЗДЕСЬ

## Что это?

**Telegram-бот нутрициолог** с искусственным интеллектом (Google Gemini).

Отправьте фото еды → получите анализ калорий, состава и рекомендации.

## ⚡ Быстрый старт (3 шага)

### 1️⃣ Получите API ключи

**Telegram Bot Token:**
- Откройте @BotFather в Telegram
- Отправьте `/newbot`
- Скопируйте токен

**Gemini API Key:**
- Откройте https://makersuite.google.com/app/apikey
- Создайте ключ
- Скопируйте ключ

### 2️⃣ Запустите на сервере Ubuntu 22.04

```bash
cd callbot
chmod +x install.sh
./install.sh
```

Введите ваши ключи когда скрипт попросит.

### 3️⃣ Готово!

Найдите вашего бота в Telegram и отправьте `/start`

## 🎯 Что умеет бот

1. **🍽 Оценка калорийности** - калории, БЖУ, вес порции
2. **📋 Анализ состава** - расшифровка этикеток, вредные добавки
3. **🥗 Оценка блюда** - сбалансированность, рекомендации
